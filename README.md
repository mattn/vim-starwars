# vim-starwars

StarWars on Vim

![vim-starwars](https://raw.githubusercontent.com/mattn/vim-starwars/master/screenshot.gif)

## Usage

```
:StarWars
```

Specify episode

```
:StarWars sw1
```

## Episodes

* SW1
* SW2

## Requirements

Vim
